## mtool/extensions

This tree stores durable companion code that must live under the `mtool/` namespace.

- `mtool/extensions/{project_key}/{source_output_key}/`
- Use: wrappers, collaborators, helpers, policies, bootstrap glue
- Do not copy raw Mtool-generated output here
- Current raw output stays under `work/source-outputs/...`
- Durable sample inputs stay under `sample/<category>/<pack>/...`
