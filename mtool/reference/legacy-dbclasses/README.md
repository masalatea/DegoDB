# Legacy DB Classes Reference

This directory stores a curated legacy DB class reference for comparison and migration context.

- It is a limited reference under `mtool/reference/`.
- It is not a current runtime input.
- It is not a replacement for generated runtime artifacts under `mtool/reference/dbclasses/`.
- Full legacy source snapshots, when present, remain host-side reference only and must not be mounted into the current Docker runtime as `original-codes/`.
