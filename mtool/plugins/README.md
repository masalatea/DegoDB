# Mtool Plugins / Mtool plugins

This directory is the planned home for packaged Mtool extension surfaces.

Current first slice:

- `ai/`: AI-facing plugins. These are instruction, schema, task-packet, example, validator, and handoff packages for Codex / Claude style workflows.

Not defined here yet:

- code-facing generator hooks;
- runtime plugin loading;
- dependency installation;
- external app or game-engine project generation.

See [AI-Facing Plugin Interface](../../docs/ai-facing-plugin-interface.md) for the current interface.
